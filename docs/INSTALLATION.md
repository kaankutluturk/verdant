# EcoAI Installation Guide

## Quick Install

```bash
git clone https://github.com/yourusername/ecoai.git
cd ecoai
pip install -r requirements.txt
python ecoai.py --setup
python ecoai.py --interactive
```

## System Requirements
- **Python**: 3.8 or higher
- **RAM**: 8GB minimum
- **Storage**: 5GB free space
